# 🚀 React Learning Journey

This repository contains my practice projects and exercises while learning **React**. It focuses on understanding core concepts through hands-on coding.

---

## 📚 Topics Covered

* JSX (JavaScript XML)
* Functional Components
* Props
* Nested Components
* Rendering Lists with `.map()`
* Conditional Rendering
* Basic Project Structure with Vite

---

## 🛠️ Technologies Used

* React (Modern React 18+)
* Vite
* JavaScript (ES6+)
* HTML & CSS

---

## 📂 Project Examples

### 🚌 Bus Dashboard App

* Displays a list of buses
* Shows routes and availability
* Uses components, props, and conditional rendering

### 🎓 Student Score Dashboard

* Displays student scores
* Shows pass/fail status
* Practices list rendering and component structure

---

## 🎯 Goals

* Understand how React works
* Build reusable components
* Practice writing clean and structured code
* Prepare for more advanced topics like state and events

## ▶️ Getting Started

To run this project locally:

```bash
npm install
npm run dev
```

Then open:

```
http://localhost:5173
```
